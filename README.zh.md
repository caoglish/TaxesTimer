# 德州扑克计时器

[English](README.md)

专为德州扑克设计的极简倒计时器，针对移动端优化。采用 7 段 LED 数码管风格显示，内置两种预设时长。

## 使用方法

### 按钮说明

| 按钮 | 颜色 | 功能 |
|------|------|------|
| 左侧（宽） | 黄色 | 开始 / 暂停 / 重置 |
| 中间 | 蓝色 | 30 秒 |
| 右侧 | 红色 | 60 秒 |

### 开始倒计时

1. 点击**蓝色**按钮设置 30 秒，或点击**红色**按钮设置 60 秒。
2. 显示屏显示所选时间后，再次点击同一按钮立即开始，或点击**黄色**按钮开始。

### 暂停与继续

- 倒计时运行中，点击**黄色**按钮**暂停**。
- 再次点击**黄色**按钮从当前位置**继续**倒计时。

### 重置

- 暂停状态下，点击**黄色**按钮重置为完整时长，再次点击重新开始。
- 倒计时归零（**00**）后，点击**黄色**按钮重置为上次使用的时长。
- 任意时刻点击**蓝色**或**红色**按钮，可立即切换到新时长并停止当前计时。

### 声效

- 每次按下按钮播放**点击音**。
- 第一秒及每隔 10 秒播放**滴答声**。
- 倒计时归零时播放**警报声**。

> 移动端首次使用需点击任意按钮以初始化音频系统。

## 开发

```bash
npm install
npm run dev      # 启动开发服务器
npm run build    # 生产构建
npm run preview  # 预览构建产物
```

在同一 Wi-Fi 下用手机测试：

```bash
npm run dev -- --host
```

用手机浏览器打开终端显示的网络地址即可。
